# 🌸 Sakura Bio Website

A cute bubbly aesthetic profile website with sakura petals and soft pink design.

## 💖 Features
- Floating sakura petals animation
- Glassmorphism UI (bubbly cards)
- Anime-inspired aesthetic
- Click sparkle effects
- Mobile friendly design

## 🖼 Preview
Displays a soft pink profile card with anime aesthetic background vibes.

## 🚀 How to use
1. Clone repo
2. Open `index.html`
3. Or deploy via GitHub Pages / Netlify
